# TinyMail
Webapplikation für Serienmails

# Über TinyMail
TinyMail ist eine Webapplikation zur bequemen und einfachen Versendung von Serienmails in 3 kleinen Schritten.

# Lizenz
TinyMail ist lizensiert unter LGPLv3.
